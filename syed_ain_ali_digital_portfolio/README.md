## 🚀 How to Deploy on GitHub Pages

1. Go to your repository settings on GitHub: `https://github.com/syedainaliaskari512-ui/syed-ain-ali-portfolio/settings/pages`
2. Under **Source**, select `Deploy from a branch`.
3. Set the branch to `main` / `root` and click **Save**.
4. Your website will be live at:
   `https://syedainaliaskari512-ui.github.io/syed-ain-ali-portfolio/`
